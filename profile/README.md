- Кот-д'Ивуар забил в **91% всех своих матчей на ЧМ** — наивысший показатель среди всех сборных с 3+ играми

## Тактический анализ

**Кот-д'Ивуар** атакует через фланги — **Амад Диалло** (Манчестер Юнайтед) справа и **Симон Адингра** (Брайтон) слева. Ян Диомандé создал **7 голевых шансов** за два матча — лучший показатель среди всех игроков. Кессье и Фофана — стержень. Задача: сломить плотный блок Кюрасао.

**Кюрасао** будет играть **5-2-3** с глубоким блоком и ставкой на контратаки через Тахита Чонга (МЮ). Именно сжатая оборона позволила держать 0:0 против Эквадора. Рум снова будет героем — но дважды повторить 15 сейвов против более качественной атаки КДИ крайне сложно.
